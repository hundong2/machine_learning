# 주요 AI 알고리즘과 그 활용 사례(Top AI Algorithms And Their Use-Cases)

**"주요 AI 알고리즘과 그 활용 사례(Top AI Algorithms And Their Use-Cases)"**를 잘 요약한 인포그래픽입니다.
사용자님은 아키텍트이자 개발자이시므로, 단순한 정의를 넘어 **알고리즘의 학습 방식(지도/비지도/강화)**과 기술적 특징을 기준으로 분류하여 상세히 설명해 드리겠습니다.

## 1. 지도 학습 (Supervised Learning)

데이터에 정답(Label)이 있어, 입력(X)과 출력(Y)의 관계를 학습하여 예측하거나 분류하는 알고리즘들입니다.

### 회귀 (Regression) - 연속적인 값 예측

#### Linear Regression (선형 회귀)
* **설명**: 데이터 포인트들 사이의 관계를 가장 잘 설명하는 직선(선형 방정식)을 찾습니다. 가장 기본적이고 해석하기 쉽습니다.
* **활용**: 주택 가격 예측 (평수, 위치 등 특성을 기반으로 가격 산출).

#### Random Forest (랜덤 포레스트)
* **설명**: 여러 개의 의사결정 나무(Decision Tree)를 만들고, 그 결과들을 종합(앙상블)하여 예측 성능을 높이고 과적합(Overfitting)을 방지합니다.
* **활용**: 주가 예측 (금융 데이터의 변동성을 여러 트리의 합의로 예측).

#### Gradient Boosting (그레디언트 부스팅)
* **설명**: 이전 모델이 범한 오류(잔차)를 다음 모델이 순차적으로 수정해 나가는 방식입니다. (예: XGBoost, LightGBM). 예측 성능이 매우 뛰어납니다.
* **활용**: 신용 점수 평가 (대출 승인 여부 판단 시 높은 정확도 요구).

### 분류 (Classification) - 카테고리 구분

#### Logistic Regression (로지스틱 회귀)
* **설명**: 이름은 회귀지만 분류 알고리즘입니다. 데이터가 특정 클래스에 속할 확률을 시그모이드(Sigmoid) 함수를 통해 0과 1 사이의 값으로 반환합니다.
* **활용**: 스팸 메일 분류 (스팸일 확률이 50% 넘으면 스팸함으로 이동).

#### Decision Trees (의사결정 나무)
* **설명**: 스무고개처럼 "Yes/No" 질문을 반복하며 데이터를 가지치기해 나갑니다. 결과의 과정을 눈으로 보기 쉽습니다(Explainable).
* **활용**: 고객 이탈 예측 (특정 행동 패턴을 보인 고객이 떠날지 남을지 분류).

#### Naive Bayes (나이브 베이즈)
* **설명**: 베이즈 통계 정리(조건부 확률)를 기반으로 합니다. 모든 특성이 독립적이라고 가정하고 계산하므로 속도가 매우 빠릅니다.
* **활용**: 텍스트 분류 (뉴스 기사 카테고리 분류, 감정 분석 등).

#### SVM (Support Vector Machines, 서포트 벡터 머신)
* **설명**: 데이터를 구분하는 가장 넓은 도로(여유 마진이 큰 결정 경계)를 찾습니다. 고차원 데이터 분류에 강합니다.
* **활용**: 필기체 인식 (숫자 이미지 분류).

#### KNN (K-Nearest Neighbors, K-최근접 이웃)
* **설명**: "유유상종" 알고리즘입니다. 새로운 데이터가 들어오면 가장 가까운 K개의 이웃을 찾아 다수결로 분류합니다. 별도의 학습 과정이 거의 없습니다(Lazy learning).
* **활용**: 영화 추천 시스템 (나와 취향이 비슷한 유저들이 본 영화 추천).

## 2. 비지도 학습 (Unsupervised Learning)

정답 없이 데이터 자체의 패턴이나 구조를 찾아내는 알고리즘들입니다.

### 클러스터링 (Clustering) - 그룹화

#### K-Means Clustering (K-평균 군집화)
* **설명**: 데이터를 K개의 그룹으로 묶습니다. 각 그룹의 중심점(Centroid)을 이동시키며 최적의 그룹을 찾습니다.
* **활용**: 고객 세분화 (마케팅을 위해 구매 패턴이 비슷한 고객끼리 그룹핑).

#### GMM (Gaussian Mixture Model, 가우시안 혼합 모델)
* **설명**: 데이터가 여러 개의 정규 분포(가우시안 분포)를 따른다고 가정합니다. K-Means보다 유연하게(확률적으로) 그룹을 나눕니다.
* **활용**: 이상 탐지 (네트워크 보안에서 일반적인 분포를 벗어난 트래픽 탐지).

### 차원 축소 및 연관 규칙

#### PCA (Principal Component Analysis, 주성분 분석)
* **설명**: 고차원 데이터(변수가 많은 것)를 정보 손실을 최소화하면서 저차원으로 압축합니다. 데이터 시각화나 노이즈 제거에 씁니다.
* **활용**: 이미지 압축 (핵심 특징만 남기고 용량 줄임).

#### Association Rule Learning (연관 규칙 학습)
* **설명**: "A를 산 사람은 B도 사더라"는 규칙을 찾습니다 (예: Apriori 알고리즘).
* **활용**: 장바구니 분석 (맥주와 기저귀의 연관성 분석 등 리테일 분야).

## 3. 딥러닝 & 신경망 (Deep Learning & Neural Networks)

인간의 뇌 구조를 모방하여 복잡한 패턴(이미지, 텍스트 등)을 학습합니다.

#### Neural Networks (신경망/ANN)
* **설명**: 입력층, 은닉층, 출력층으로 구성된 기본 인공 신경망입니다.
* **활용**: 얼굴 인식 (이미지의 픽셀 패턴 학습).

#### RNN (Recurrent Neural Networks, 순환 신경망)
* **설명**: 이전 단계의 출력이 다음 단계의 입력으로 들어가는 고리 구조를 가집니다. 순서가 있는 데이터(시계열, 텍스트) 처리에 적합합니다.
* **활용**: 감정 분석 (문맥을 파악하여 긍정/부정 판단).

#### LSTM (Long Short-Term Memory)
* **설명**: RNN의 단점(오래된 기억을 잊어버리는 문제)을 해결하여 장기 기억을 가능하게 한 모델입니다.
* **활용**: 주식 시장 예측 (과거의 긴 데이터를 바탕으로 추세 예측).

#### Word Embeddings (워드 임베딩)
* **설명**: 단어를 숫자로 된 벡터로 변환하여 단어 간의 의미적 유사성을 컴퓨터가 이해하게 합니다 (예: Word2Vec).
* **활용**: 검색 엔진 관련성 향상 (키워드가 정확히 일치하지 않아도 의미가 유사하면 검색됨).

#### NLP (Natural Language Processing, 자연어 처리)
* **참고**: NLP는 알고리즘이라기보다 분야에 가깝습니다. 위에서 언급한 RNN, LSTM, Transformer 등을 사용하여 언어를 처리합니다.
* **활용**: 고객 지원 챗봇.

## 4. 기타 / 최적화 알고리즘

#### Genetic Algorithms (유전 알고리즘)
* **설명**: 생물의 진화 과정(선택, 교차, 돌연변이)을 모방하여 최적의 해를 찾습니다. 수학적으로 풀기 너무 복잡한 문제에 사용합니다.
* **활용**: 공급망 최적화 (가장 효율적인 물류 경로 찾기).

#### Ant Colony Optimization (개미 군집 최적화)
* **설명**: 개미가 페로몬을 남겨 최단 경로를 찾는 방식을 모방합니다. 그래프 최단 경로 문제에 강합니다.
* **활용**: 외판원 문제(TSP) (여러 도시를 가장 짧게 방문하는 경로 찾기).

#### Reinforcement Learning (강화 학습)
* **설명**: 에이전트가 환경과 상호작용하며 보상(Reward)을 최대화하는 방향으로 학습합니다. 시행착오를 겪으며 배웁니다.
* **활용**: 게임 플레이 (알파고, 자율주행, 로봇 제어).
