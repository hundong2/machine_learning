# 고려대학교 인공지능 강의

# Basic of Machine Learning 

- 기계학습의 정의
  - 명시적인 규칙 없이 데이터 간 관련성을 스스로 찾아 낼 수 있는 컴퓨터 프로그램

- 성능평가 지표 P를 개선하기 위한 방법 : 데이터의 개수를 늘린다. 

## Supervised Learning 

- 지도 학습
  - 데이터가 입력(특징벡터)과 출력(목표 값) 쌍으로 구성되어 있다. 
  - 지도 학습 중 회귀(Regression)는 출력이 연속된 실수로 주어진다.
  - 지도 학습의 출력 값은 주어진 입력에 대해 항상 정답으로 간주된다. 

- 예제 : 자연어 번역

- Calssification(분류)를 활용한 문제 : 스팸 이메일, 기사의 작성자 예측, 질병 종류 예측 등

## Unsupervised Learning

- 비지도 학습
  - 주어진 입력(특징 벡터)에 대해서 출력(목표 값이)이 제공 되지 않는다. 
  - 대표적인 비지도 학습 기법으로 군집화 ( Clustering)는 전체 데이터를 몇 개의 세부 그룹을 나누어 구성한다. 
  - 대표적인 비지도 학습 기법으로 차원 축소 ( Dimensionality Reduction )는 고차원의 입력 데이터를 저차원으로 특징 벡터로 변환하여 표현한다.

- 군집화( Clustering )
  - 데이터 사이의 거리 함수의 정의에 따라 군집의 결과는 달라질 수 있다. 
  - 미리 지정된 클러스터(cluster)의 개수의 따라 군집의 결과는 달라질 수 있다.
  - 클러스터(cluster)의 수가 증가할수록 데이터 간 평균 거리는 감소하는 경향이 있다. 

## Reinforcement Learning

- 강화학습 ( Reinforcement Learning )
  - 학습 후, 주어진 상태(state)에 대한 최적의 행동(action)을 예측할 수 있다.
  - 선택 가능한 행동(action) 중 누적된 보상(reward)를 최대화하는 행동을 선택한다.
  - 보상을 제공하는 환경(environment)는 마르코프(markov)결정 과정으로 표현된다.

- 예제 : 제약된 환경에서의 제어이론, 시뮬레이션 기반 최적화, 규칙이 정해진 게임 최적화 등...

- Deep neural networks 
  - 특징
    - 입력 변수들 간의 비선형 조합이 가능.
    - 특징 표현(Feature representation)이 학습 과정에서 함께 수행
    - 데이터가 증가함에 따라 성능이 개선
  - 성공한 이유?
    - 데이터 증가, 다수의 파라미터 학습을 위한 하드웨어 증가, 과적합 문제를 해결을 위한 다양한 규제화(Regularization) 기법으로 인해 심층 신경망(DNN)이 성공
