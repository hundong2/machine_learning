# ResNet 논문 한국어 의역 자료

> 이 문서는 논문 원문 전체 번역이 아니라, 초보자가 논문의 흐름을 잡기 위한 한국어 의역과 해설입니다. 정확한 실험 조건, 표, 수식은 [원문](https://arxiv.org/abs/1512.03385)을 기준으로 확인하세요.

## 논문 제목 의역

**이미지 인식을 위한 깊은 잔차 학습**

## 한 문장 요약

층을 더 깊게 쌓아도 정보와 기울기가 흐르도록 입력을 다음 층에 더해 주는 스킵 커넥션을 제안했습니다.

## 초록을 초보자 말로 풀어쓰기

단순히 층을 깊게 만들면 학습 오차가 오히려 나빠지는 degradation 문제가 나타납니다. 이는 모델 표현력 부족보다 최적화가 어려운 문제에 가깝습니다.

블록이 원하는 함수 H(x)를 직접 배우는 대신 F(x)=H(x)-x를 배우고, 출력은 F(x)+x로 만듭니다. 필요 없는 블록은 0에 가까운 잔차만 내면 항등 함수처럼 행동할 수 있습니다.

수십 층을 넘어 152층 수준까지 안정적으로 학습하며 ImageNet과 detection/segmentation 전반의 표준 백본이 되었습니다.

## 핵심 주장 3개 의역

1. 이 논문은 기존 방법이 놓치던 병목을 문제로 정의합니다. 먼저 "왜 이 문제가 그 시점에 중요했는가"를 잡으면 좋습니다.
2. 제안 방법의 힘은 복잡한 장식보다 구조적 관점의 변화에서 나옵니다. 즉, 모델이 어떤 정보를 더 쉽게 보거나 전달하도록 길을 바꾼 것입니다.
3. 실험 결과는 단순히 점수가 좋아졌다는 뜻만이 아니라, 이후 연구자들이 어떤 방향으로 모델을 키우고 조합할지에 대한 기준점을 만들었습니다.

## 처음 읽을 때 표시할 부분

- 문제 정의: 논문이 무엇을 실패 사례로 보고 있는지 찾습니다.
- 모델 그림: 데이터가 어떤 모양으로 들어가 어떤 모양으로 나오는지 따라갑니다.
- 손실 함수 또는 학습 목표: 모델이 무엇을 맞히도록 훈련되는지 확인합니다.
- 실험 표: 단순 최고 점수보다, 어떤 조건을 바꿨을 때 성능이 바뀌는지 봅니다.

## 연구자처럼 던질 질문

- 이 논문의 가정은 지금도 유효한가?
- 모델이 잘되는 이유를 저자들은 실험으로 어떻게 분리했는가?
- 내가 더 작은 장난감 데이터로 같은 현상을 재현한다면 무엇을 측정해야 하는가?
