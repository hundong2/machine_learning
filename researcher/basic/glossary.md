# Glossary

논문을 읽다가 자주 막히는 단어를 빠르게 풀어보는 작은 사전입니다.

| 용어 | 쉬운 설명 | 연구자 관점에서 중요한 이유 |
|---|---|---|
| 합성곱 | 작은 필터를 이미지 위에 반복 적용해 국소 패턴을 찾는 연산 | 같은 특징 탐지기를 위치마다 공유하므로 파라미터가 줄고 위치 변화에 강해집니다. |
| 필터/커널 | 이미지의 작은 영역을 보는 숫자 격자 | 어떤 필터는 세로선, 어떤 필터는 모서리처럼 특정 패턴에 반응합니다. |
| 풀링 | 작은 영역을 대표값 하나로 줄이는 다운샘플링 | 위치가 조금 바뀌어도 비슷한 특징으로 남게 해줍니다. |
| 활성화 함수 | 선형 계산 뒤에 넣는 비선형 함수 | 층을 여러 개 쌓아도 단순 선형식 하나로 무너지지 않게 만듭니다. |
| ReLU | 0보다 작으면 0, 크면 그대로 통과시키는 함수 | 계산이 단순하고 양수 영역에서 기울기가 잘 살아 학습이 빨라집니다. |
| 기울기 소실 | 역전파 중 기울기가 너무 작아 앞쪽 층이 거의 학습하지 못하는 현상 | 깊은 신경망 학습이 어려워지는 핵심 이유입니다. |
| 스킵 커넥션 | 입력을 몇 개 층 뒤 출력에 바로 더하는 길 | 정보와 기울기가 우회로를 통해 더 잘 흐릅니다. |
| 패치 | 이미지를 작은 정사각형 조각으로 자른 것 | ViT와 MAE에서 이미지의 단어 역할을 합니다. |
| 토큰 | 모델이 한 단위로 처리하는 벡터 | 텍스트의 단어, 이미지의 패치가 모두 토큰이 될 수 있습니다. |
| 위치 인코딩 | 토큰에 원래 위치 정보를 더하는 값 | 패치를 한 줄로 세우면 위치를 잃기 때문에 필요합니다. |
| Self-Attention | 토큰끼리 서로 얼마나 참고할지 계산하는 연산 | 멀리 떨어진 이미지 영역의 관계를 직접 볼 수 있습니다. |
| 임베딩 | 데이터를 숫자 벡터 좌표로 바꾼 표현 | 이미지와 텍스트를 같은 공간에서 비교할 수 있게 합니다. |
| 코사인 유사도 | 두 벡터 방향이 얼마나 비슷한지 재는 값 | CLIP에서 이미지와 텍스트가 의미상 맞는지 비교할 때 자주 씁니다. |
| 대조 학습 | 맞는 쌍은 가깝게, 틀린 쌍은 멀게 학습하는 방식 | 라벨 대신 이미지-텍스트 쌍 같은 자연 데이터로 학습할 수 있습니다. |
| 자기지도 학습 | 데이터 자체에서 문제와 정답을 만들어 학습하는 방식 | 사람 라벨 없이도 대규모 표현 학습이 가능합니다. |
| 오토인코더 | 입력을 압축했다가 다시 복원하는 모델 | 잠재 공간과 복원 학습을 이해하는 기본 구조입니다. |
| 잠재 공간 | 원본보다 작고 압축된 특징 공간 | Latent Diffusion은 이 공간에서 확산을 수행해 계산 비용을 낮춥니다. |
| 정방향 확산 | 깨끗한 이미지에 점점 노이즈를 더하는 과정 | 역방향 복원을 학습하기 위한 훈련 문제를 만듭니다. |
| 역방향 확산 | 노이즈에서 이미지를 점점 복원하는 과정 | 생성 과정의 핵심입니다. |

## 코드에서 자주 만나는 용어

| 용어 | 쉬운 설명 | 노트북에서 보는 위치 |
|---|---|---|
| `shape` | 배열의 생김새입니다. 예를 들어 `(16, 16)`은 16행 16열 이미지입니다. | 모든 노트북에서 입력과 출력 크기를 확인할 때 |
| `axis` | 계산 방향입니다. `axis=0`은 세로 방향, `axis=1`은 가로 방향으로 생각하면 출발하기 쉽습니다. | `mean`, `sum`, `softmax`, `norm` |
| `reshape` | 같은 숫자들을 다른 모양의 배열로 다시 보는 연산입니다. 숫자 개수는 변하지 않습니다. | patch를 한 줄 벡터로 펼칠 때 |
| `@` | 행렬 곱 연산자입니다. 신경망의 선형층, attention의 점수 계산에 자주 나옵니다. | AlexNet, ViT, CLIP 실습 |
| `np.maximum(x, 0)` | ReLU를 numpy로 쓴 형태입니다. 0보다 작은 값은 0으로 바꿉니다. | LeNet-5, AlexNet, ResNet 실습 |
| `softmax` | 여러 점수를 확률처럼 합이 1이 되게 바꾸는 함수입니다. | ViT attention, CLIP loss |
| `temperature` | softmax에 들어가기 전 점수를 얼마나 날카롭게 만들지 조절하는 값입니다. 작을수록 가장 큰 점수에 확률이 몰립니다. | CLIP 실습 |
| `alpha_bar` | diffusion에서 원본 이미지가 현재 단계에 얼마나 남아 있는지 나타내는 값입니다. | Latent Diffusion 실습 |
| `mask_ratio` | 전체 패치 중 가릴 패치 비율입니다. 0.75는 75%를 가린다는 뜻입니다. | MAE 실습 |
| `keepdims=True` | 합이나 평균을 낸 뒤에도 차원 모양을 유지하게 하는 옵션입니다. broadcasting 실수를 줄입니다. | softmax, 정규화 함수 |
| broadcasting | 크기가 다른 배열끼리 계산할 때 numpy가 자동으로 모양을 맞추는 규칙입니다. | 정규화, bias 더하기 |
