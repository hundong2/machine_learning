# Masked Autoencoder (MAE) 핵심 자료

## 문제

좋은 비전 모델을 만들려면 많은 라벨이 필요했습니다. 하지만 라벨링은 비싸고, 인터넷 이미지 대부분은 정답지가 없습니다.

## 제안 아이디어

이미지 패치의 75% 정도를 가린 뒤, encoder는 보이는 패치만 처리하고 가벼운 decoder가 전체 이미지를 복원합니다.

## 결과와 의의

ViT 기반 모델의 자기지도 사전학습을 효율적으로 만들었고, 대규모 라벨 없는 이미지 학습의 중요한 레시피가 되었습니다.

## 핵심 개념

- 자기지도 학습
- 마스킹
- 오토인코더
- 비대칭 encoder-decoder
- 복원 손실

## 수식 직관

- 입력을 `x`, 모델이 배우려는 변환을 `f`라고 보면, 연구의 핵심은 `f(x)` 자체보다 `x`가 어떤 표현으로 바뀌어야 학습이 쉬운지를 설계하는 데 있습니다.
- 좋은 논문은 대개 파라미터를 많이 늘리는 것만이 아니라, 정보가 더 잘 흐르는 좌표계나 경로를 제안합니다.
- 실습 노트북에서는 논문 전체 모델을 그대로 재현하지 않고, 이 직관을 가장 작은 계산 단위로 분해합니다.

## 코드와 연결하기

- 배열의 shape을 먼저 봅니다. 비전 모델은 거의 항상 `높이 x 너비 x 채널` 또는 `배치 x 채널 x 높이 x 너비`의 형태 문제입니다.
- 함수 하나를 볼 때는 입력 shape, 출력 shape, 학습되는 값인지 고정된 계산인지부터 구분합니다.
- 결과 그림은 예쁜 시각화가 아니라, 논문 아이디어가 정말 작동하는지 확인하는 측정 도구로 봅니다.

## 헷갈리기 쉬운 지점

- 논문 모델 전체를 처음부터 구현하려고 하면 막히기 쉽습니다. 먼저 한 블록의 입출력만 이해해도 충분합니다.
- 성능 수치는 데이터셋, 학습 시간, 전처리, 모델 크기까지 함께 봐야 합니다.
- 유명한 논문일수록 후속 연구의 개선점이 많습니다. 여기서는 "처음 문제를 어떻게 열었는가"에 집중합니다.

## 다음으로 이어지는 생각
