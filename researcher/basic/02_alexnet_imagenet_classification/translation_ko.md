# AlexNet 논문 한국어 의역 자료

> 이 문서는 논문 원문 전체 번역이 아니라, 초보자가 논문의 흐름을 잡기 위한 한국어 의역과 해설입니다. 정확한 실험 조건, 표, 수식은 [원문](https://papers.nips.cc/paper/4824-imagenet-classification-with-deep-convolutional-neural-networks)을 기준으로 확인하세요.

## 논문 제목 의역

**깊은 합성곱 신경망을 이용한 ImageNet 분류**

## 한 문장 요약

CNN을 대규모 데이터, GPU, ReLU, 드롭아웃과 결합해 현대 딥러닝 붐을 폭발시킨 논문입니다.

## 초록을 초보자 말로 풀어쓰기

ImageNet 같은 실사 이미지는 손글씨 숫자보다 훨씬 복잡합니다. 기존 특징 공학과 작은 CNN만으로는 1000개 클래스를 안정적으로 구분하기 어려웠습니다.

큰 CNN을 GPU로 학습하고, 포화되기 쉬운 tanh 대신 ReLU를 사용하며, 드롭아웃과 데이터 증강으로 과적합을 줄입니다.

ILSVRC 계열 이미지 분류에서 기존 접근과 큰 격차를 보이며 딥러닝 기반 컴퓨터 비전의 전환점을 만들었습니다.

## 핵심 주장 3개 의역

1. 이 논문은 기존 방법이 놓치던 병목을 문제로 정의합니다. 먼저 "왜 이 문제가 그 시점에 중요했는가"를 잡으면 좋습니다.
2. 제안 방법의 힘은 복잡한 장식보다 구조적 관점의 변화에서 나옵니다. 즉, 모델이 어떤 정보를 더 쉽게 보거나 전달하도록 길을 바꾼 것입니다.
3. 실험 결과는 단순히 점수가 좋아졌다는 뜻만이 아니라, 이후 연구자들이 어떤 방향으로 모델을 키우고 조합할지에 대한 기준점을 만들었습니다.

## 처음 읽을 때 표시할 부분

- 문제 정의: 논문이 무엇을 실패 사례로 보고 있는지 찾습니다.
- 모델 그림: 데이터가 어떤 모양으로 들어가 어떤 모양으로 나오는지 따라갑니다.
- 손실 함수 또는 학습 목표: 모델이 무엇을 맞히도록 훈련되는지 확인합니다.
- 실험 표: 단순 최고 점수보다, 어떤 조건을 바꿨을 때 성능이 바뀌는지 봅니다.

## 연구자처럼 던질 질문

- 이 논문의 가정은 지금도 유효한가?
- 모델이 잘되는 이유를 저자들은 실험으로 어떻게 분리했는가?
- 내가 더 작은 장난감 데이터로 같은 현상을 재현한다면 무엇을 측정해야 하는가?
