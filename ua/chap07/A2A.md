## install agent inspect

```sh
git remote -v
origin	https://github.com/a2aproject/a2a-inspector.git (fetch)
origin	https://github.com/a2aproject/a2a-inspector.git (push)
cd a2a-inspector
uv sync
cd frontend 
npm install 

npm run build #front end file build
cd ../backend
uv run app.py
```
