### 🚀 1. 로지스틱 회귀의 3단계 파이프라인 수식

데이터 $\mathbf{x}$가 들어와서 최종 벌점 $L$이 계산되기까지, 수식은 정확히 3단계를 거칩니다.

#### **[1단계] 선형 결합 (Linear Combination): "데이터의 총점 구하기"**
가장 먼저, 입력된 데이터에 가중치를 곱하고 편향을 더해 하나의 숫자(총점)로 만듭니다.
$$z = \mathbf{w}^T\mathbf{x} + b$$
* $\mathbf{w}$: 가중치 벡터 (각 특징의 중요도)
* $\mathbf{x}$: 입력 데이터 벡터 (예: 공부 시간, 출석 횟수 등)
* $b$: 편향 (기본 점수)
* $z$: 산출된 총점 (범위: $-\infty \sim +\infty$)

#### **[2단계] 시그모이드 (Sigmoid Function): "확률로 변환하기"**
1단계에서 구한 총점 $z$를 $0$과 $1$ 사이의 확률값으로 찌그러뜨립니다. 이것이 모델의 '최종 예측'입니다.
$$p = \sigma(z) = \frac{1}{1 + e^{-z}}$$
* $e$: 자연상수 (약 2.718)
* $p$: 모델이 예측한 성공(1) 확률 (범위: $0 \sim 1$)
* **의미:** $z$가 $0$일 때 $p=0.5$가 되며, $z$가 클수록 $p$는 $1$에, 작을수록 $p$는 $0$에 수렴합니다.

#### **[3단계] 로그 손실 (Log Loss / Cross-Entropy): "채점하고 벌점 매기기"**
모델이 예측한 확률 $p$와 실제 정답 $y$를 비교하여 오차(벌점)를 계산합니다.
$$L(w, b) = - \left[ y \log(p) + (1-y) \log(1-p) \right]$$
* $y$: 실제 정답 ($1$ 또는 $0$)
* **의미:** 정답이 $1$일 때는 앞부분 $-y \log(p)$만 남고, 정답이 $0$일 때는 뒷부분 $-(1-y) \log(1-p)$만 남아서 완벽하게 오차를 계산해 내는 천재적인 스위치 수식입니다.

### 🌍 2. 전체 비용 함수 (Cost Function): "최종 성적표"

위의 3단계는 **'데이터 1개'**에 대한 채점 과정이었습니다.
실제 학습에서는 전체 데이터($m$개)에 대한 평균 벌점을 구해야 하는데, 이를 손실(Loss)과 구분하여 **비용 함수(Cost Function)** $J(w, b)$라고 부릅니다.

$$J(w, b) = - \frac{1}{m} \sum_{i=1}^{m} \left[ y^{(i)} \log(p^{(i)}) + (1-y^{(i)}) \log(1-p^{(i)}) \right]$$
* $m$: 전체 데이터의 개수
* $\sum$: 모든 데이터의 벌점을 다 더한다는 기호
* **목표:** 경사 하강법은 바로 이 거대한 U자형 밥그릇인 **$J(w, b)$를 가장 작게 만드는(최소화) 가중치 $w$와 편향 $b$를 찾는 알고리즘**입니다.

### 🤯 3. 궁극의 단일 수식 (합체)

만약 전공 서적이나 논문에서 로지스틱 회귀의 전체 손실 함수를 단 한 줄로 길게 써놓았다면, 그것은 위에서 배운 3개의 수식을 러시아 인형(마트료시카)처럼 겹겹이 끼워 넣은 것에 불과합니다.

$p$ 자리에 시그모이드 식을 넣고, 그 안의 $z$ 자리에 선형 결합 식을 집어넣으면 다음과 같은 무시무시한(하지만 이제는 친숙한) 전체 수식이 완성됩니다.

$$L(w, b) = - \left[ y \log\left(\frac{1}{1 + e^{-(\mathbf{w}^T\mathbf{x} + b)}}\right) + (1-y) \log\left(1 - \frac{1}{1 + e^{-(\mathbf{w}^T\mathbf{x} + b)}}\right) \right]$$

### 💡 한눈에 보는 요약 치트시트

| 단계 | 역할 | 수식 (Formula) | 의미 |
| :--- | :--- | :--- | :--- |
| **입력** | **선형 결합** | $z = wx + b$ | 데이터를 가중치와 곱해 $z$ (총점) 생성 |
| **예측** | **시그모이드** | $p = \frac{1}{1 + e^{-z}}$ | $z$를 $0 \sim 1$ 사이의 확률 $p$로 변환 |
| **평가** | **로그 손실** | $L = -[y \log(p) + \dots]$ | 예측 확률 $p$와 정답 $y$의 차이(오차) 계산 |
| **학습** | **도함수(미분)** | $\frac{\partial L}{\partial w} = (p - y) \cdot x$ | 오차를 줄이기 위해 $w$를 어느 방향으로 바꿀지 계산 |
| **수정** | **경사 하강법** | $w_{new} = w_{old} - \alpha \frac{\partial L}{\partial w}$ | 계산된 기울기 방향으로 학습률 $\alpha$만큼 $w$ 업데이트 |
