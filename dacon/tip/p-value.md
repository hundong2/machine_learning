# p-value와 귀무가설/대립가설
p-value는 귀무가설이 맞다고 가정했을 때, 지금처럼 극단적인 결과가 나올 확률입니다.

## 예시 파일
14.1.2.DateTimePandas2.ipynb

## 답변
질문하신 내용을 아주 간단히 정리하면 이렇습니다.

- **귀무가설 $H_0$**: 보통 “차이 없음”, “효과 없음”, “0”에 해당합니다.
- **대립가설 $H_1$**: 보통 “차이 있음”, “효과 있음”, “1”에 해당합니다.
- **p-value가 작다**: 귀무가설이 맞다고 보기 어렵다.
- **p-value가 0.05 이하**: 보통 **귀무가설을 기각**합니다.

즉, **p-value ≤ 0.05라고 해서 “positive(1)다”라고 직접 말하는 건 아니고**,  
정확히는 **“귀무가설을 버리고 대립가설 쪽 증거가 있다”**는 뜻입니다.

### 예시
이진 분류처럼 생각하면 이렇게 볼 수 있습니다.

- $H_0$: positive가 아니다
- $H_1$: positive다

이때 p-value가 0.03이면:

- 귀무가설이 맞다는 가정하에 이런 결과가 나올 확률이 3%
- 너무 작으니 귀무가설을 기각
- 그래서 **positive라고 볼 근거가 있다**고 해석할 수 있음

### 중요한 주의점
- **p-value는 “대립가설이 맞을 확률”이 아닙니다.**
- **p-value ≤ 0.05 = 무조건 positive**는 아닙니다.
- 항상 **귀무가설과 대립가설을 무엇으로 정의했는지** 먼저 봐야 합니다.

### ADF 테스트에 적용하면
시계열 정상성 검정에서는 보통:

- **귀무가설 $H_0$**: 비정상 시계열이다
- **대립가설 $H_1$**: 정상 시계열이다

따라서 p-value ≤ 0.05이면:

- **비정상이라는 귀무가설을 기각**
- 즉 **정상 시계열일 가능성이 높다**고 해석합니다.
