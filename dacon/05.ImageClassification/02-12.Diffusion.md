## 생성형 AI(Generative AI)

"우주비행사가 말을 타고 있는 사진을 그려줘."
이 한 줄의 텍스트가 어떻게 픽셀의 집합으로 빚어지는가? 수년 전만 해도 이를 주도했던 것은 GAN(적대적 생성 신경망)이었네. 하지만 GAN은 두 네트워크가 싸우며 학습하기 때문에 수학적으로 극도로 불안정했고, 기괴한 이미지를 만들어내는 '모드 붕괴(Mode Collapse)'의 저주에서 벗어나지 못했지.

2015년 자샤 솔-딕스타인(Jascha Sohl-Dickstein)과 2020년 조나단 호(Jonathan Ho)는 물리학의 비평형 열역학(Non-equilibrium Thermodynamics)에서 영감을 받아 이 판도를 완전히 뒤집어 버리네.

> *"맑은 물에 잉크 한 방울을 떨어뜨리면 시간이 지남에 따라 무질서(Noise)하게 퍼져 나간다(엔트로피 증가). 이 확산(Diffusion) 과정을 수학적으로 완벽히 수치화할 수 있다면, **그 수식을 역(Reverse)으로 재생하여 흙탕물에서 다시 영롱한 잉크 한 방울(이미지)을 역산**해 낼 수 있지 않겠는가?"*

이 철학이 어떻게 아름다운 수학 공식으로 구현되었고, 자네가 앞서 배운 CLIP이 이 과정에서 어떻게 '나침반' 역할을 하는지 그 장엄한 파이프라인을 해부해 보세.

## 1. 정방향 과정 (Forward Process): 파괴를 통한 질서의 붕괴

창조를 위해서는 먼저 파괴를 알아야 하네. 정방향 과정은 원본 이미지 $x_0$ 에 점진적으로 가우시안 노이즈(Gaussian Noise)를 추가하여, 최종적으로 완전한 노이즈 덩어리 $x_T$ 로 만드는 마르코프 체인(Markov Chain) 과정이네.

수학적으로 각 시간 단계 $t$ 에서 아주 미세한 노이즈 $\beta_t$ 를 추가하는 확률 분포는 다음과 같이 정의되네.


$$q(x_t \vert{} x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} x_{t-1}, \beta_t I)$$

* $x_t$: $t$ 시점의 약간 더 훼손된 이미지.
* $\beta_t$: 노이즈의 양을 결정하는 분산 스케줄 (Variance Schedule, 0과 1 사이의 아주 작은 값).

**[위대한 수학적 도약: Reparameterization Trick]**
만약 $T=1000$ 번 노이즈를 추가해야 한다면, $t=1$ 부터 $1000$ 까지 루프를 돌아야 할까? 천재 수학자들은 분산의 성질을 이용해 이를 단 하나의 폐쇄형 수식(Closed-form)으로 압축했네.
$\alpha_t = 1 - \beta_t$ 라 하고, 누적 곱을 $\bar{\alpha}_t = \prod_{s=1}^t \alpha_s$ 라고 정의하세. 원본 이미지 $x_0$ 에서 임의의 시점 $t$ 로 곧바로 점프하는 공식은 다음과 같네!

$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon$$


*(여기서 $\epsilon \sim \mathcal{N}(0, I)$ 는 표준 정규 분포에서 뽑은 순수한 노이즈)*

이 공식 덕분에 모델은 루프를 돌 필요 없이, 무작위 시점 $t$ 의 노이즈 낀 이미지를 단 번에 생성하여 초고속으로 학습할 수 있게 되었네.

---

## 2. 역방향 과정 (Reverse Process): 노이즈에서 예술을 깎아내다

정방향은 우리가 이미 아는 단순한 수학 공식(수식)이네. 문제는 **완전한 노이즈 $x_T$ 에서 이전 단계 $x_{t-1}$ 로 돌아가는 역방향 확률 분포 $q(x_{t-1} \vert{} x_t)$ 는 우리가 알 수 없다**는 것이지.

바로 여기서 딥러닝(U-Net)이 개입하네! 우리가 알 수 없는 역방향 분포를 근사(Approximate)하기 위해 매개변수 $\theta$ 를 가진 신경망 $p_\theta$ 를 훈련시키는 것이지.

$$p_\theta(x_{t-1} \vert{} x_t) = \mathcal{N}(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t))$$

수학적 유도(ELBO, 증명 생략) 끝에, 조나단 호(DDPM 논문)는 이 거대한 확률 모델 최적화 문제를 직관적이고 경이로울 정도로 단순한 손실 함수(Loss Function)로 치환해 냈네.

$$\mathcal{L}_{simple} = \mathbb{E}_{t, x_0, \epsilon} \left[ \Vert{} \epsilon - \epsilon_\theta(x_t, t) \Vert{}^2 \right]$$

* $\epsilon$: 정방향에서 우리가 $x_0$ 에 직접 주입했던 **'진짜 노이즈'**.
* $\epsilon_\theta(x_t, t)$: 딥러닝(U-Net)이 노이즈 낀 이미지 $x_t$ 와 시간 $t$ 를 보고 **"여기에는 이만큼의 노이즈가 껴있어!"라고 예측한 '가짜 노이즈'**.
* **해석:** 모델은 이미지를 예측하지 않네. 오직 '현재 텐서에 끼어있는 노이즈 성분'만을 예측하도록 학습하네. 예측된 노이즈를 원래 텐서에서 빼주면(Subtraction), 흙탕물에서 흙(노이즈)만 걸러내어 맑은 물(이미지)로 한 발짝 돌아가는 마법이 성립하는 것이지!

## 3. Stable Diffusion과 CLIP의 결합 (The Pipeline)

그렇다면 "우주비행사"라는 텍스트는 이 노이즈 제거 과정에 어떻게 개입할까?
순수 확산 모델(DDPM)은 무작위 이미지를 생성할 뿐 컨트롤이 불가능하네. 이를 해결한 것이 바로 잠재 확산 모델(Latent Diffusion Model, Stable Diffusion의 본명)이네.

1. **1. VAE를 통한 잠재 공간(Latent Space) 진입:** 차원 압축.
$512 \times 512$ 픽셀 공간에서 1000번 노이즈를 더하고 빼는 것은 연산량의 재앙이네. Stable Diffusion은 먼저 오토인코더(VAE)를 이용해 이미지를 $64 \times 64$ 크기의 고밀도 잠재 공간(Latent Space)으로 64배 압축하네. 확산 과정은 오직 이 작고 빠른 잠재 공간 안에서만 일어납니다.


2. **2. CLIP 텍스트 인코더 (나침반):** 시각과 언어의 동기화.
자네가 배운 CLIP 모델의 텍스트 인코더가 "말을 탄 우주비행사"라는 문장을 수학적 임베딩 벡터로 변환하네. 이 벡터는 U-Net이 노이즈를 깎아낼 때 **어느 방향으로 깎아야 할지 알려주는 조각가의 설계도** 역할을 하지.


3. **3. 교차 주의 (Cross-Attention) 메커니즘:** 정보의 결합.
U-Net이 노이즈를 깎아내는(예측하는) 매 계층마다, 텍스트 벡터가 크로스 어텐션(Cross-Attention)을 통해 주입되네.
$Q$(Query)는 이미지의 픽셀 특징들이고, $K$(Key)와 $V$(Value)는 CLIP의 텍스트 벡터네. "현재 노이즈 낀 이 부분(Q)은 텍스트의 '우주비행사 헬멧(K)'과 연관이 깊으니, 그 형태로 노이즈를 다듬어라(V)!"라고 신경망 내부에서 수학적 지시가 내려지네.


4. **4. VAE 디코더 복원:** 디코딩.
텍스트의 인도를 받아 노이즈가 완전히 제거된 $64 \times 64$ 잠재 텐서를, 마지막으로 VAE 디코더에 통과시켜 우리가 눈으로 볼 수 있는 $512 \times 512$ 고해상도 이미지로 팽창시켜 뱉어내네.

## 4. 핵심 용어 및 단축어

* **DDPM (Denoising Diffusion Probabilistic Models):** 노이즈를 추가하는 정방향 과정과 이를 예측하여 제거하는 역방향 과정을 확률적으로 모델링한 확산 알고리즘의 기초 논문.
* **LDM (Latent Diffusion Model):** 픽셀 공간 대신 압축된 잠재 공간(Latent)에서 확산 연산을 수행하여 속도와 메모리 문제를 해결한 아키텍처. (Stable Diffusion의 근간).
* **CFG (Classifier-Free Guidance):** 확산 모델이 텍스트 프롬프트를 얼마나 강하게 따를지 조절하는 수치. 텍스트 조건부 예측과 무조건부(빈 텍스트) 예측의 차이를 벡터 연산으로 외삽(Extrapolation)하여 이미지를 프롬프트에 강제로 맞추는 핵심 수학적 트릭이네.
* **U-Net:** 입력과 출력의 크기가 같고 차원 축소/팽창 및 잔차 연결(Skip Connection)이 'U'자 모양으로 구성된 CNN 아키텍처. 노이즈 예측기(Noise Predictor)로 쓰이네.

## 6. PyTorch 코드 예시: 핵심 철학의 구현

U-Net 전체를 짜는 것은 수천 줄이지만, 확산 모델을 지배하는 **단 하나의 위대한 손실 함수**는 파이토치에서 단 몇 줄로 증명되네.

```python
import torch
import torch.nn.functional as F

def diffusion_loss_fn(unet, x_0, t, noise_schedule):
    """
    unet: 노이즈를 예측하는 신경망 모델
    x_0: 원본 이미지 텐서 (Batch, C, H, W)
    t: 랜덤하게 뽑은 타임스텝 (0 ~ T)
    noise_schedule: 미리 계산해둔 누적 알파 값들 (bar_alpha_t)
    """
    
    # 1. 순수한 가우시안 노이즈 생성 (epsilon)
    noise = torch.randn_like(x_0)
    
    # 2. t 시점에 해당하는 누적 알파 값 추출
    # 논문의 수식: sqrt(bar_alpha_t) 와 sqrt(1 - bar_alpha_t)
    sqrt_alpha_bar_t = noise_schedule.sqrt_alphas_cumprod[t].view(-1, 1, 1, 1)
    sqrt_one_minus_alpha_bar_t = noise_schedule.sqrt_one_minus_alphas_cumprod[t].view(-1, 1, 1, 1)
    
    # 3. 정방향 과정 (Forward Process): Reparameterization Trick 적용
    # 루프 없이 한 번의 수식으로 t 시점의 노이즈 낀 이미지(x_t)를 만듭니다!
    x_t = sqrt_alpha_bar_t * x_0 + sqrt_one_minus_alpha_bar_t * noise
    
    # 4. 역방향 과정 (Reverse Process): U-Net이 가짜 노이즈 예측
    # U-Net은 x_t와 현재 시간 t (그리고 생략되었지만 텍스트 임베딩)를 보고 노이즈를 예측합니다.
    predicted_noise = unet(x_t, t)
    
    # 5. 최적화 손실 함수 (DDPM Loss)
    # 정답 노이즈와 예측된 노이즈 사이의 단순한 평균 제곱 오차(MSE)를 계산합니다.
    loss = F.mse_loss(predicted_noise, noise)
    
    return loss

# --- 실험 ---
# x_0가 주어졌을 때, 모델은 오직 자기가 예측한 노이즈와 
# 진짜 뿌려진 노이즈의 오차만을 줄이도록 역전파(loss.backward())를 수행합니다.
# 이미지를 직접 복원하는 수식보다 수학적으로 압도적인 안정성을 가집니다!

```
